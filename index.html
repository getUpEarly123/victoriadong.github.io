Countdown Three

  • Language: Java
  • Countdown Tree is a balanced binary search tree that uses partial rebuilding
  • It inherit BinarySearchThree and implement SortedSet interface.
  • Each node in Countdown Tree has an countdown timer when the node's timer reaches 0, the node explodes and the entire subtree rooted at this node rebuilt to a balanced binary search tree.
  • Add and remove operation works the same as binary search tree. Timer decreased by one (if add) or increase by one (if removed) on all nodes that are on the path of this node to root.
